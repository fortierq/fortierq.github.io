Simplexe à deux phases

Cas simple

Pour commencer l'algorithme du simplexe, il faut partir d'une base admissible (valeurs des variables qui vérifient toutes les contraintes, les variables hors base étant égales à 0).
Souvent, on peut utiliser les variables d'écarts comme base initiale. Considérons par exemple le PL ci-dessous:

{max⁡z=4x+5y2x+y≤4x+2y≤2x,y≥0\left\{ \begin{array}{ll} \max z = 4x + 5y\\ 2x + y \leq 4 \\ x + 2y \leq 2 \\ x, y \geq 0 \end{array} \right. ⎩⎪⎪⎪⎨⎪⎪⎪⎧​maxz=4x+5y2x+y≤4x+2y≤2x,y≥0​

Il peut être mis sous forme standard en ajoutant des variables d'écarts :

{max⁡z=4x+5y2x+y+e1=4x+2y+e2=2x,y,e1,e2≥0\left\{ \begin{array}{ll} \max z = 4x + 5y\\ 2x + y + e_1 = 4 \\ x + 2y + e_2 = 2 \\ x, y, e_1, e_2 \geq 0 \end{array} \right. ⎩⎪⎪⎪⎨⎪⎪⎪⎧​maxz=4x+5y2x+y+e1​=4x+2y+e2​=2x,y,e1​,e2​≥0​

(e1,e2)(e_1, e_2)(e1​,e2​) forme une base admissible car, si x=y=0x = y = 0x=y=0, on trouve e1=4e_1 = 4e1​=4, e2=2e_2 = 2e2​=2 qui sont bien positifs (donc vérifient les contraintes x,y,e1,e2≥0x, y, e_1, e_2 \geq 0x,y,e1​,e2​≥0). On peut donc, dans ce cas, appliquer l'algorithme du simplexe avec (e1,e2)(e_1, e_2)(e1​,e2​) comme base initiale.

Cas problématique

Cependant, parfois il est plus difficile de trouver une première base admissible. Considérons le programme linéaire suivant, légèrement modifié par rapport au précédent :

(∗){max⁡z=4x+5y−2x+y+e1=−4x−2y+e2=−2x,y,e1,e2≥0(*) \left\{ \begin{array}{ll} \max z = 4x + 5y\\ -2x + y + e_1 = -4 \\ x - 2y + e_2 = -2 \\ x, y, e_1, e_2 \geq 0 \end{array} \right. (∗)⎩⎪⎪⎪⎨⎪⎪⎪⎧​maxz=4x+5y−2x+y+e1​=−4x−2y+e2​=−2x,y,e1​,e2​≥0​

Trouver une première base admissible

Si x=y=0x = y = 0x=y=0 alors e1=−4e_1 = -4e1​=−4 et e2=−2e_2 = -2e2​=−2, ce qui n'est pas une base admissible (car négatifs). La méthode du simplexe à 2 phases consiste à modifier le PL en ajoutant des variables artificielles a1a_1a1​, a2a_2a2​ :

(∗∗){min⁡y=a1+a2−2x+y+e1−a1=−4x−2y+e2−a2=−2x,y,e1,e2,a1,a2≥0(**)\left\{ \begin{array}{ll} \min y = a_1 + a_2\\ -2x + y + e_1 - a_1 = -4 \\ x - 2y + e_2 - a_2 = -2 \\ x, y, e_1, e_2, a_1, a_2 \geq 0 \end{array} \right. (∗∗)⎩⎪⎪⎪⎨⎪⎪⎪⎧​miny=a1​+a2​−2x+y+e1​−a1​=−4x−2y+e2​−a2​=−2x,y,e1​,e2​,a1​,a2​≥0​

On a fait exprès de mettre des signes −-− devant a1a_1a1​ et a2a_2a2​ pour qu'ils constituent une base admissible de ce nouveau PL (∗∗)(**)(∗∗). En effet, si x=y=e1=e2=0x = y = e_1 = e_2 = 0x=y=e1​=e2​=0 alors a1=4a_1 = 4a1​=4 et a2=2a_2 = 2a2​=2, qui sont bien positifs. L'idée est ensuite d'utiliser l'algorithme du simplexe sur ce nouveau PL pour sortir a1a_1a1​ et a2a_2a2​ de la base. On obtiendra alors une base constituée uniquement d'éléments parmi x,y,e1,e2x, y, e_1, e_2x,y,e1​,e2​, qui sera aussi une base du PL initial (∗)(*)(∗).
